JavaScript Basics

Java script is a cross-platform, object-oriented scripting language. JavaScript is extremeley popular for a variety of reasons. It is a small and lightweight language allowing maximum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host environment (a web browswer or Node server), it can be connected to the objects of these environments to provide programmatic control over them.

  • Variable Declaration JavaScript variables are containers... These unique names are called identifieers. var x;
  • Variable Assignment Assignment operators assign values to... the = assignemnt operator assigns a value to a variable. var x = 10;
  • Data Types Data types are an important concept. There are six data types that are JavaScript primitives: Boolean - true or false;... null aka nothing; Number - 42 or 3.14159; String - "Coding Dojo Rocks!"; Array - [1, 'Coding', 2, 'Dojo']; and Object - {first_name: 'Jane', last_name:'Doe'};
  • Conditionals When you write code, you want to blah blah blah. There are if a specified condition is true, do this code in our code block; else if to specify a new condition to test, if the first codition is false; else we execute this block of code;
